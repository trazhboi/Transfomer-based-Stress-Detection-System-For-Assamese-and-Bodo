# Multilingual Stress Detection from Speech Signals Using ML Models
### Low-Resource Languages — Assamese & Bodo
 
---
 
## Overview
 
This project presents a machine learning pipeline for detecting lexical and prosodic stress from speech signals in two low-resource Northeast Indian languages — *Assamese* and *Bodo*. Given the scarcity of labeled speech data for these languages, we adopt a dual-phase approach: classical ML models on a small hand-recorded dataset, followed by transformer-based fine-tuning on a larger dataset harvested from Assamese movie audio.
